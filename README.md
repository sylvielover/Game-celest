# Proyecto organizado por capas

## Estructura

- index.html → estructura principal
- assets/
  - css/
    - styles.css → estilos del juego
  - js/
    - game.js → lógica del juego

## Ventajas
- Código más limpio
- Fácil mantenimiento
- Separación entre estructura, estilos y lógica
- Escalable para agregar más niveles y funcionalidades
